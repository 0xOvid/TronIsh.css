TronIsh.css

A design system for creating cool futuristis lookung UIs.


Intro

TronIsh.css is a CSS libray to style web GUIs, The library is meant to make it easy to quickly create good looking webinterfaces for my various projects.

Card

The "card" element allows you to create "boxes" with various content. Each of which acts like a column in a row, the row being the "grid element". Furthermore to get the full experince the following elements also need to be placed in order: cardHeader, cardBorder, and cardContent.

HEADER
Cool contents

Code:


<div class="card">
    <div class="cardHeader">HEADER</div>
    <div class="cardBorder">
    <div class="cardContent">
        Cool contents
    </div>
</div>

Cards can also be "selected", highlighting the header element.

SELECTED HEADER
Cool contents

Code:


<div class="card">
    <div class="cardHeaderSelected">SELECTED HEADER</div>
    <div class="cardBorder">
    <div class="cardContent">
        Cool contents
    </div>
</div>
